# CodeAlpha Calculator

This is a basic calculator web application created as part of the CodeAlpha Frontend Development Internship.

## Project Description

The calculator is built using HTML, CSS, and JavaScript. It performs basic arithmetic operations and provides a clean, responsive user interface.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Percentage operation
- Clear button
- Delete button
- Decimal support
- Keyboard support
- Error handling
- Responsive design
- Hover effects and modern UI

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

1. Open the calculator in your browser.
2. Click the buttons to enter numbers and operators.
3. Press `=` to calculate the result.
4. Press `C` to clear the screen.
5. Press `DEL` to delete the last value.
6. You can also use your keyboard.

## Keyboard Controls

- Numbers: `0-9`
- Operators: `+`, `-`, `*`, `/`, `%`
- Calculate: `Enter`
- Delete: `Backspace`
- Clear: `Escape`
